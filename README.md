# Simon Says Game

An interactive memory game built using **HTML, CSS, and JavaScript**. Players must repeat an increasingly complex sequence of colors to advance through levels.

## Features
- Random sequence generation
- Level progression
- Score tracking
- Interactive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Run Locally
1. Clone the repository.
2. Open `index.html` in your browser.
